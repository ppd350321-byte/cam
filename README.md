# 社区食堂营运管理系统-前台管理端  
***Community Canteen Operation & Supply Chain Management System - Admin Web***

## 项目介绍  
基于Vue.js开发的后台管理系统，供食堂管理员进行订单管理、生产调度、供应链配置、报表查看、用户权限管控等操作，适配PC端大屏展示。  

## 核心功能模块  
- **仪表盘**：实时营运数据概览（订单量、营收、生产进度）  
- **用户管理**：VIP客户信息、积分/预存账户操作  
- **订单管理**：订单审核、生产派单、支付状态跟踪  
- **供应链配置**：供货商管理、原材料采购计划生成  
- **生产调度**：分钟级任务编排、任务完成状态监控  
- **报表中心**：盈亏分析、绩效统计、产品销量趋势图  
- **系统设置**：权限配置、支付接口管理、优惠规则设置  


## 毕设题目

**《社区食堂的营运及其购销链条管理系统设计》**

从社区食堂的个体客户下订、支付，到终端订单转化为原材料如肉蛋菜的供货商，店内的准确到分钟的生产调度，统一通过一个web来实现。其中包括：

- 数据库，变成，网站实现。
- 个体客户的VIP，积分，优惠，预存，奖励，支付接口安排。
- 以及企业的营运报表生成，雇员的绩效评价，形成一个有效的可靠的专有的营运系统。
- 同时对安全性进行加强、测试，评价。

基于提前的订单，智能地自动处理人工分配，任务指定，材料采购计划，产品定时定点生产，任务完成后的营业信息分析，比如个人的贡献，企业盈亏，某产品类型增加或减少订单量，哪些采用限制性设置，支付方式，预存支付成功后，账户的VIP管理等。
